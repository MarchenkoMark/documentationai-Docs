## Overview

Quickly add new documentation to your project.

## Steps

<Steps>
  <Step title="Go to Projects">
    Navigate to the Projects page in the main menu.
  </Step>
  <Step title="Click 'New Doc'">
    Press the <b>New Doc</b> button to start a new document.
  </Step>
  <Step title="Fill in details">
    Enter a title and content for your documentation page. Save when finished.
  </Step>
</Steps>
