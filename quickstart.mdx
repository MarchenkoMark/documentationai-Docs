## Prerequisites

Before you begin, ensure you have:

- A valid email address for account verification
- Access to a modern web browser like Chrome or Firefox

<Callout kind="info">

Mark Marchenko works best on desktop browsers. Mobile support is available but limited for editing.

</Callout>

## Create Your Account

Follow these steps to sign up and verify your email.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit [https://markmarchenko.com](https://markmarchenko.com) and click **Sign Up**.

    Fill in your email, create a strong password, and submit the form.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification email from Mark Marchenko.

    Click the verification link to activate your account.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the site and log in with your verified credentials.

    You'll land on the dashboard automatically.
  </Step>
</Steps>

## Set Up Your First Documentation Space

Spaces organize your project docs like folders.

<Steps>
  <Step title="Create Space" icon="folder-plus">
    Click **New Space** in the sidebar.

    Enter a name like `My Project Docs` and select a color (e.g., brand blue `#3B82F6`).
  </Step>
  <Step title="Add First Document" icon="file-plus">
    Inside the space, click **New Document**.

    Title it `README.mdx` and start editing.
  </Step>
  <Step title="Publish" icon="globe">
    Click **Publish** to make it live.

    Share the generated URL with your team.
  </Step>
</Steps>

<Callout kind="tip">

Use the preview pane to see changes in real-time as you edit MDX content.

</Callout>

## Import Existing Documents

Bring in your current docs seamlessly.

<Tabs>
  <Tab title="Markdown Files" icon="file-text">
    Drag and drop `.md` or `.mdx` files into your space.

    Or use the import button and select files from your drive.

````markdown
```mdx
# Welcome to My Docs

This is imported content.
```
````
  </Tab>
  <Tab title="Google Docs" icon="file">
    Export your Google Doc as Markdown, then import as above.

    Preserve formatting with tools like [markdown export](https://docs.google.com).
  </Tab>
  <Tab title="GitHub Repo" icon="github">
    Connect your GitHub account via settings.

    Import entire repos as spaces automatically.
  </Tab>
</Tabs>

## Basic Navigation and Search Tips

Master these for efficient workflows.

<Columns cols={2}>
  <Card title="Keyboard Shortcuts" icon="keyboard">
    - `Cmd/Ctrl + K`: Quick search
    - `Cmd/Ctrl + P`: Command palette
    - `Esc`: Focus search bar
  </Card>
  <Card title="Advanced Search" icon="search">
    Use filters like `space:project inurl:readme` to find docs fast.
  </Card>
  <Card title="Star Favorites" icon="star">
    Star important docs for sidebar pinning.
  </Card>
  <Card title="Version History" icon="git-branch">
    Right-click documents for diffs and restores.
  </Card>
</Columns>
